信件模块

全局描述

主要是跟信件相关的操作

全局权限说明

字段 来源 类型 默认值 描述 验证规则 忽略Action
random HEADER string - 随机数8位数组 required: 必须填写
-
imei HEADER string - 设备唯一标识 required: 必须填写
-
times HEADER string - 10时间戳 required: 必须填写
-
sign HEADER string - 加密签名 required: 必须填写
-
from_type HEADER string - 请求来源（ios,android,pc_h5,wx_h5,wap_h5,wx_miniapp） required: 必须填写
-
language HEADER string - 语言(cn,en,hk) required: 必须填写
-
token HEADER 默认 - 没有传空字符串 required: 必须填写
-

统一电子信写信

基本信息

Path： /letter/virtual

Method： GET,POST

接口描述： 统一电子信接口

请求字段

字段 来源 类型 默认值 描述 验证规则
letter_type 不限 int - 1邮箱2微信3微信小程序4qq5抖音小程序6快手小程序7支付宝小程序 required: 必须填写
content 不限 string - 信件内容 required: 必须填写
open 不限 int - 是否公开 required: 必须填写
email 不限 string - 收信邮箱 required: 必须填写
send_time 不限 string - 收信时间 -
order 不限 string - 关联订单 -
upload_url 不限 string - 音频路径 -
image_url 不限 string - 照片路径 -
send_email 不限 string - 发信邮箱 -
stamp_id 不限 string - 邮票id -

请求示例

请求示例1

curl http://10.0.0.15:9501/letter/virtual

成功响应示例

成功响应示例1

{
    "code": 200,
    "data": {
        "login_user_info": {
            "headimgurl": "",
            "nickname": "李四",
            "last_login_time": 1655888389
        },
        "token": {
            "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9",
            "expirationTime": 1656493189
        },
        "ref_token": {
            "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9",
            "expirationTime": 1671440389
        }
    },
    "msg": "操作成功"
}

失败响应示例

失败响应示例1

    "code": 400,
    "data": null,
    "msg": "字段非法"
}

统一登录入口

基本信息

Path： /user/login

Method： GET,POST

接口描述： 登录入口

请求字段

字段 来源 类型 默认值 描述 验证规则
auth_type 不限 string app_weixin app_weixin(app微信登录)
app_qq(appQQ登录)
miniapp_weixin(微信小程序登录)
h5_weixin(微信H5登录)
 required: 必须填写
lengthMax: 长度不能超过'20'
unionid GET string - unionid required: 必须填写
lengthMax: 长度不能超过'50'
openid 不限 string - openid required: 必须填写
lengthMax: 长度不能超过'50'
nickname 不限 string - 昵称 lengthMax: 长度不能超过'50'
headimgurl 不限 string - 头像 lengthMax: 长度不能超过'100'
province 不限 string - 城市 lengthMax: 长度不能超过'20'
city 不限 string - 地区 lengthMax: 长度不能超过'20'
from_type 不限 string - 登录类型（ios,android,h5,pc） lengthMax: 长度不能超过'20'

curl http://10.0.0.15:9501/user/login
